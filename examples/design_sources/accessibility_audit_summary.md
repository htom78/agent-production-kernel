# Accessibility Audit Summary

Source: codex/skills/accessibility-audit.md from
https://github.com/Trystan-SA/claude-design-system-prompt
License: MIT

Structured checks mapped into APK:

- Contrast and color, including WCAG AA ratios.
- Semantic HTML and heading structure.
- Keyboard reachability, logical tab order, and visible focus.
- Motion, forms, error linkage, input types, and touch target size.
