# Claude Design System Prompt Summary

Source: https://github.com/Trystan-SA/claude-design-system-prompt
License: MIT

APK uses this as design-domain knowledge, not as a global operating prompt.
Relevant extracted principles:

- Start from available product, brand, code, screenshot, or UI-kit context.
- Treat generic AI aesthetics as a failure mode.
- Prefer structured components and tokens over one-off visual choices.
- Verify rendered output before declaring design work complete.
- Preserve accessibility, interaction states, hierarchy, and polish as gates.
